## Project: SMART_IOT

Back-end IOT using NodeJS with Node.js, Express, MongoDB  build with fully restfull API

## DOCUMENT

ExpressJS: https://expressjs.com/ 
MongoDB: https://mongoosejs.com/

## HOW TO SETUP

Clone: git clone 
cd back_end
npm install

Create .env file
add: PORT:{yourPort}
    MONGO_URI:{yourDatabase}

Run project: npm run dev
